欢迎来到"数字信号处理实验室"的文档页面。

我们已经默认你早已了解此应用的用处，想必你也大概知道我们将要用到的技术。不过，鉴于科研小组的新成员对这些不是很了解，在此处我还是列出这个应用所用到的技术。

网页前端(html+css+js)
网页后台(php+mysql+apache)
web服务器搭建(apache+mysql+php)
matlab web server服务的配置
matlab中m文件的编写

对于服务器的配置您不必自己研究，可以找许剑帮你配置好。

网页前端

前端没什么可以多说的，毕竟我们重点不是在前端。我们只追求也面的简洁，功能的完善。

网页后台

web服务器我们使用php+mysql+apache架构，而matlab web server安装在另外一台机器上，这台机器上也安装有web服务器作为一个文件服务器，因为matlab处理后的图片等文件存在matlab web server的机器上。在这里我们使用codeigniter(CI，一种php框架)框架，这是一款小巧但是效率很高的框架。相比yii，CI更符合我们的要求。

Matlab部分

Matlab在这里是核心，没有m文件就不能处理任何数据。这里面你需要了解matlab指令以及m文件的编写，对于信号处理的一些知识也可以了解下，这对于以后的工作会很有帮助。

普通的m文件编写很简单，但是这里需要运用到matlab web server中，因此，编写还是有些要注意的。您可以点击这里，学习该应用中m文件的编写。如果您只负责matlab部分，那你可以不用理会其他，只需要看着部分就行了。
